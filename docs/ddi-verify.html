แต่แอพจะเอาตาราง ddi_class_rules จาก Supabase มาแทนที่ค่า default ทั้งชุด — และตารางนั้นน่าจะมีข้อมูลไม่ครบ (สาเหตุที่เป็นไปได้มากสุด: การ Import Defaults รุ่นแรก เคยมีบั๊ก keyword ซ้ำ ทำให้ import ล้มกลางคัน/ได้บางส่วน — บั๊กนั้นแก้แล้วใน v5.51.0) ยากลุ่ม opioid/benzodiazepine จึงหายไปจากชุดที่ใช้จริงบนหน้าเว็บ → ยืนยันได้ด้วย SQL ในหัวข้อ 5)

วิธีแก้ (หลังตรวจข้อมูลในเอกสารนี้เสร็จ): เข้า admin → แท็บ DDI → กด "Import Defaults" อีกครั้ง (เวอร์ชันปัจจุบันจะ merge/update รายการเดิมให้เอง ไม่สร้างซ้ำ) แล้วข้อมูลครบชุดล่าสุดจะถูก sync ขึ้น Supabase
จุดที่ควรพิจารณาเป็นพิเศษ (ผมติดธงไว้ให้):
1. cnsDepress ตอนนี้ตั้งระดับเป็น Moderate — แต่ UpToDate/Lexicomp จัด Opioid + Benzodiazepine เป็น Risk D / Major และ US FDA มี Boxed Warning สำหรับคู่นี้ → ควรพิจารณาปรับเป็น Major (เหตุที่เดิมตั้ง moderate: ใน ICU มักตั้งใจใช้ร่วมภายใต้ monitor เช่น fentanyl+midazolam sedation — ถ้าปรับเป็น major ข้อความ "จัดการ" ยังอธิบายบริบท ICU ไว้อยู่)
2. dexmedetomidine ติดแค่ bradycardia — เป็นยา sedative ควรติด cnsDepress ด้วยไหม?
3. ketamine ติด cnsDepress ไว้ (กดหายใจน้อยกว่ากลุ่มอื่น) — คงไว้หรือถอด?
4. tramadol/pethidine + ondansetron จะชนกันผ่าน class serotonergic — สอดคล้อง Lexicomp (Risk C)
5. ยาในฐานข้อมูล compat ที่ยังไม่ติด tag ใด ๆ เช่น olanzapine (QT/CNS), quetiapine ไม่มีในระบบ IV — ช่วยดูหัวข้อ 2) ว่ามียาที่ใช้บ่อยในโรงพยาบาลคุณตกหล่นไหม

1) นิยาม Class (10 classes) — ระดับความรุนแรง / กลไก / การจัดการ

Class = กลุ่มความเสี่ยงสะสม: ถ้าเลือกยา ≥2 ตัวที่อยู่ class เดียวกัน ระบบจะเตือนอัตโนมัติ

✓Classระดับกลไกการจัดการอ้างอิง
💓 QT
QT prolongation (เสี่ยง Torsades)		 รุนแรง (Major)ยาหลายตัวที่ยืด QT ใช้ร่วมกัน → ยืด QTc เสริมกัน → เสี่ยง Torsades de Pointesเลี่ยงใช้ร่วมถ้าทำได้; ตรวจ ECG (baseline + ระหว่างให้), แก้ K⁺/Mg²⁺ ให้ปกติ, ระวังในผู้สูงอายุ/หัวใจCredibleMeds; Lexicomp
🧠 serotonergic
Serotonin syndrome		 รุนแรง (Major)ยา serotonergic หลายตัวร่วมกัน → serotonin มากเกิน → ไข้สูง, สั่น, clonus, สับสนเฝ้าระวังอาการ (clonus/hyperthermia/agitation); เลี่ยงใช้ร่วมหลายตัว โดยเฉพาะกับ MAOI/linezolidLexicomp
🫘 nephrotoxic
Additive nephrotoxicity (พิษต่อไต)		 รุนแรง (Major)ยาพิษต่อไตหลายตัวร่วมกัน → เสริมความเสียหายต่อไตติดตาม SCr/UO ทุกวัน, ให้สารน้ำเพียงพอ, เลี่ยงใช้ร่วมถ้าทำได้, ปรับขนาดตามไตLexicomp
🩸 bleeding
Additive bleeding risk (เลือดออก)		 รุนแรง (Major)ยาที่เพิ่มความเสี่ยงเลือดออก (anticoagulant/antiplatelet/NSAID/thrombolytic) ร่วมกัน → เลือดออกมากขึ้นประเมินความเสี่ยง-ประโยชน์, ติดตามอาการเลือดออก/Hb, พิจารณา PPI ป้องกัน GI bleedLexicomp
⚡ hyperK
Additive hyperkalemia (K⁺ สูง)		 ปานกลาง (Moderate)ยา/สารที่เพิ่ม K⁺ ร่วมกัน → K⁺ ในเลือดสูง → เสี่ยงหัวใจเต้นผิดจังหวะติดตาม K⁺ และ ECG, ระวังในผู้ป่วยไตเสื่อมLexicomp
👂 ototoxic
Additive ototoxicity (พิษต่อหู)		 ปานกลาง (Moderate)ยาพิษต่อหู (aminoglycoside/vancomycin/loop diuretic/cisplatin) ร่วมกัน → เสริมความเสียหายต่อการได้ยิน/สมดุลติดตามการได้ยิน/อาการเวียน, เลี่ยงใช้ร่วมระยะยาว, ติดตามระดับยา (TDM)Lexicomp
😴 cnsDepress
Additive CNS/respiratory depression (กดประสาท/การหายใจ)		 ปานกลาง (Moderate)ยากดประสาทส่วนกลางหลายตัว (opioid/benzodiazepine/barbiturate/propofol/sedating antihistamine) ร่วมกัน → กดการหายใจและระดับความรู้สึกตัวเสริมกันมัก “ตั้งใจ” ใช้ร่วมใน ICU ที่ monitor — ต้องเฝ้า RR/SpO₂/sedation score, มีอุปกรณ์ช่วยหายใจ + naloxone/flumazenil พร้อม; ระวังมากนอก ICU/ผู้สูงอายุLexicomp
🐢 bradycardia
Additive bradycardia / AV block (หัวใจเต้นช้า)		 รุนแรง (Major)ยากดอัตราการเต้นหัวใจ/การนำไฟฟ้า (β-blocker + non-DHP CCB + digoxin + amiodarone + dexmedetomidine) ร่วมกัน → bradycardia/AV block รุนแรงเลี่ยง IV β-blocker ร่วม IV diltiazem/verapamil; ติดตาม HR + ECG ต่อเนื่อง, เตรียม atropine/pacingLexicomp
📉 hypotension
Additive hypotension / vasodilation (ความดันต่ำ)		 ปานกลาง (Moderate)ยาขยายหลอดเลือด/ลดความดันหลายตัว (nitrate/nitroprusside/hydralazine/dihydropyridine CCB/milrinone) ร่วมกัน → ความดันโลหิตตกเสริมกันtitrate ทีละตัว, ติดตาม BP ใกล้ชิด (มัก invasive line), ระวัง reflex tachycardiaLexicomp
🌵 anticholinergic
Additive anticholinergic burden (ฤทธิ์ต้านโคลิเนอร์จิก)		 ปานกลาง (Moderate)ยาต้านโคลิเนอร์จิกหลายตัว (atropine/glycopyrrolate/hyoscine/antihistamine/benztropine) ร่วมกัน → ปากแห้ง, ปัสสาวะคั่ง, ลำไส้ไม่เคลื่อน (ileus), สับสน/เพ้อประเมินความจำเป็น, ระวัง delirium ในผู้สูงอายุ + urinary retention/ileus, ติดตามอาการLexicomp
