🔖 Mockup: ออกแบบ Nav + Footer ใหม่

ไฟล์นี้เป็นแค่ภาพตัวอย่างเพื่อตัดสินใจ — ยังไม่ได้แตะหน้าจริง เปิดดู/กดเล่นได้ กดปุ่มขวาบนเพื่อสลับโหมดมืด

เป้าหมาย 3 ข้อ
  1. Nav แบบที่คั่นหนังสือ — ติดขอบจอ ไม่กินพื้นที่เนื้อหา เข้าทุกหน้าได้รวดเร็ว
  2. Footer เดียวใช้ทุกหน้า — กระชับเป็นแถบบาง ๆ รายละเอียดยาว (disclaimer/license) ย้ายเข้า popup
  3. เลข version จุดเดียว — แก้ที่เดียวแล้วเปลี่ยนทุกหน้า (ตอนนี้ฝังมือ 7 จุดใน HTML)

↑ เลข version ดึงจากค่าเดียว APP_VERSION — แก้ที่เดียว เปลี่ยนทุกหน้า
เบื้องหลัง — ทำยังไงให้ footer/nav/version เป็น "จุดเดียว"

สร้างไฟล์ใหม่ js/site-chrome.js ตัวเดียว ทำหน้าที่ฉีด nav + footer เข้าทุกหน้า:

  • แต่ละหน้า HTML เหลือแค่ที่ว่าง 1 จุด เช่น <div id="siteFooter"></div> → JS เติมให้
  • เลข version อยู่ในค่าเดียว const APP_VERSION = '5.15.3' (หรืออ่านจาก version.json)
  • ลบ footer ยาว ๆ ที่ copy ซ้ำออกจาก 8 หน้า — เหลือ source เดียว
  • เข้ากับ build.js ได้ (inline JS ตอน deploy เหมือนไฟล์ js อื่น)

เทียบงานเภสัชฯ: เหมือนทำ "แม่แบบฉลากยา" แผ่นเดียว แล้วทุกซองดึงไปใช้ — แก้แม่แบบที่เดียว ทุกซองเปลี่ยนตาม ไม่ต้องไล่แก้ทีละซอง
